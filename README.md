# Culinaire — Your Personal Chef App

## Deploy to Vercel (Free — 2 minutes)

### Step 1: Push to GitHub
1. Go to [github.com/new](https://github.com/new) and create a new repo called `culinaire`
2. Unzip this project folder on your computer
3. Open Terminal and run:
   ```
   cd culinaire
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/culinaire.git
   git push -u origin main
   ```

### Step 2: Deploy on Vercel
1. Go to [vercel.com](https://vercel.com) and sign up with your GitHub account
2. Click **"Add New Project"**
3. Import your `culinaire` repo
4. Vercel auto-detects Vite — just click **"Deploy"**
5. In ~60 seconds you'll get a live URL like `culinaire.vercel.app`

### Step 3: Add to Your iPhone Home Screen
1. Open the Vercel URL in **Safari** on your iPhone
2. Tap the **Share button** (square with arrow)
3. Scroll down and tap **"Add to Home Screen"**
4. Name it **Culinaire** and tap **Add**

It will now appear as a full-screen app on your home screen — no browser bar, looks and feels native.
